> 평가지표 / 타깃 / 마감일은 대회 페이지에서 확인 후 `docs/overview.md`에 채우기.

```
data/               대회 데이터
data/notebooks/     참고용 상위 노트북

notebooks/          실험 노트북
submissions/        제출 파일
```

```bash
conda activate s6e5
```
